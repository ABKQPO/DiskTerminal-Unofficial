# DiskTerminal-Unofficial（元件终端）

**DiskTerminal** 模组的 1.7.10（GTNH）移植版 —— 一款用于管理 ME 网络中所有存储元件与存储总线的终端及无线终端。

> 非官方移植版。强制前置依赖：**Applied Energistics 2 Unofficial**。

[English README](README.md)

## 功能

- **终端与无线访问**：将元件终端部件安装在 ME 线缆上即可使用，或通过无线元件终端实现远程访问。与常规操作一致，需通过安全终端将无线终端与你的 ME 网络绑定。
- **多元件支持**：兼容所有 AE2 存储元件类型（物品、流体，以及通过 Thaumic Energistics 的源质）。
- **网络概览**：按位置分类展示网络中所有 ME 驱动器、ME 箱子及存储总线。
- **元件管理**：查看元件容量（已用 / 总字节数及存储类型）、直接将元件弹出至背包、预览存储物品及对应数量、通过拖拽编辑分区。
- **多种视图模式**：元件终端（简洁列表）、元件背包（网格）、元件分区（网格）、存储总线背包、存储总线分区。
- **搜索与筛选**：支持背包 / 分区 / 混合三种搜索模式，并提供高级查询语法，查找所有元件 / 存储总线中的物品。
- **世界高亮**：双击任意存储条目，即可在游戏世界中高亮对应方块，并在聊天框显示其坐标。
- **优先级管理**：直接在 GUI 中设置 ME 箱子、ME 驱动器及存储总线的优先级。
- **快速分区快捷键**：可配置快捷键，针对悬停选中的物品快速创建分区。

## 模组集成

- **Applied Energistics 2 Unofficial**（必需）
- **AE2 Fluid Craft – Rework**：流体元件与流体存储总线
- **Thaumic Energistics**：源质元件与存储总线
- **NotEnoughItems**：从物品列表拖拽至分区、快速分区
- **GregTech 5 Unofficial**：扫描 GT ME 总线 / 仓
